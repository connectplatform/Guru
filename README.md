## Testing

To run the server tests:

```bash
npm test
```
-or-
```bash
mocha test/server/specs
```

To run the client tests:

```bash
./scripts/build/traffic
mocha test/traffic/components
```

To run the integration tests:

```bash
./scripts/build/traffic
mocha test/traffic/specs
```
